Quadrado
A = l²
l = lado
Exemplo
l = 5 cm
A = 5² = 25 cm²
Todos os lados são iguais. Multiplique o lado por ele mesmo.
Retângulo
A = b × h
b = base h = altura
Exemplo
b = 8 cm, h = 5 cm
A = 8 × 5 = 40 cm²
Base vezes altura. Lados opostos são iguais.
Triângulo
A = (b × h) / 2
b = base h = altura
Exemplo
b = 10 cm, h = 6 cm
A = (10 × 6) / 2 = 30 cm²
Metade da área do retângulo de mesma base e altura.
Círculo
A = π × r²
r = raio π ≈ 3,14159
Exemplo
r = 4 cm
A = 3,14 × 4² = 3,14 × 16 = 50,24 cm²
O raio é metade do diâmetro: r = d ÷ 2.
Losango
A = (D × d) / 2
D = diagonal maior d = diagonal menor
Exemplo
D = 12 cm, d = 8 cm
A = (12 × 8) / 2 = 48 cm²
As diagonais se cruzam perpendicularmente no centro.
Trapézio
A = ((B + b) × h) / 2
B = base maior b = base menor h = altura
Exemplo
B = 12, b = 8, h = 5 cm
A = ((12+8)×5)/2 = 100/2 = 50 cm²
Média das duas bases paralelas, vezes a altura.
Triângulo Equilátero
A = (l² × √3) / 4
l = lado √3 ≈ 1,732
Exemplo
l = 6 cm
A = (36 × 1,732) / 4 = 62,35 / 4 ≈ 15,6 cm²
Todos os 3 lados iguais. Ângulos internos de 60° cada.
Hexágono Regular
A = (3√3 × l²) / 2
l = lado √3 ≈ 1,732
Exemplo
l = 4 cm
A = (3×1,732×16)/2 = 83,14/2 ≈ 41,6 cm²
6 lados iguais. Equivale a 6 triângulos equiláteros iguais.
